Printing Notes

  • Material: PETG recommended for durability (PLA works for quick prototypes).
  • Layer height: 0.16–0.24 mm (balance detail vs. speed).
  • Supports: enable only as needed (the previews help you spot overhangs).
  • If a preview is blank, open DevTools console to see load errors (often file path or CORS on some hosts).
